# CollabCanvas 🎨

CollabCanvas is a real-time collaborative whiteboard and communication platform built using the MERN stack and Socket.io. It allows multiple users to join a room, draw together on a shared whiteboard, and communicate through live chat in real time.

---

## 🚀 Features

- 🎨 Real-time collaborative whiteboard
- 💬 Live chat functionality
- 👥 Online users tracking
- ✏️ Drawing tools (Pencil, Line, Rectangle)
- 🎯 Presenter mode support
- ↩️ Undo & Redo functionality
- 🧹 Clear canvas option
- 📱 Responsive user interface
- ⚡ Real-time synchronization using Socket.io

## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/AnushaBhavani1/collabcanvas.git
```

### Install dependencies

#### Frontend

```bash
cd frontend
npm install
npm run dev
```

#### Backend

```bash
cd backend
npm install
node server.js
```
